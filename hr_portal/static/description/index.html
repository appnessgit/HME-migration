Appness Human Resource Self Service

Appness Human Resource Self Service package provides a platform for your employees to access their main transactions through portal users. Each module is directly connected to the employee data in Odoo HR.

My Profile

Allow you employees to see their information including their work information and personal data.

In employee profile, a communication panel that will directly be connected to Odoo chatter box

Enable/ Disable

Through HR settings, you have the ability to select which self service modules to be provided to your employees through the self service portal.

Leaves

A dashboard that will summarize the allocated leave balances

Employee can see the list of all their leave requests in list view and calendar view

Employee can see the status of each leave request

Employee can filter leaves either all/ approved/refused

Employee can sort leaves either by name or by the newest leave request created

Allow your employees to create leave requests

For leave types with allocation, an employee can see their remaining leave balance

If the user doesn’t have enough leave balance, blocking message will forbid the user from submitting the request

The portal mirrors the same restrictions on leave requests as Odoo leaves module

In leave request, a communication panel that will directly be connected to Odoo’s chatter box

Expenses

Employee can see the list of all their expenses

Employee can see the status of each expense

Employee can filter expenses either all/ approved/submitted

Employee can sort expenses either by name or by the newest expense created

Allow your employees to create their personal expenses and submit it

The expense is directly converted into an expense report for manager approval

Allow your employee to upload the receipt as attachment

Employee can sort expenses either by name or by the newest expense created

In expense, a communication panel that will directly be connected to Odoo’s chatter box

Payroll

Allow your employees to see the list of all their confirmed payslips

Allow the employee to directly print the payslip from the self-service portal

Allow the employee to download the payslip in the form of PDF format file

Attendance

Allow your employee to see the list of all their records

Attendance records will display the number of worked hours

Employee can filter attendance records by attendance month
